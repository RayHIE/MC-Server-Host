# MC Server Host

A comprehensive Minecraft server management application with an intuitive GUI for creating, configuring, and managing Minecraft servers.

## Requirements

For this application to work, you will need PyQt6 installed.
`pip install PyQt6`

## Features

### Current
- **Server Management**: Create and manage Minecraft servers
- **Firewall Integration**: Configure firewall rules for server access
- **Server Creator**: User-friendly interface for setting up new servers
- **Server Configuration**: Manage server settings and properties
- **Settings Panel**: Customize application preferences

### Planned
- **Fabric Server Creation**: Support for creating and managing Fabric modded servers
- **PaperMC Server Creation**: Support for creating and managing PaperMC servers
